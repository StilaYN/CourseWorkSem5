Окно авторизации

После ввода в адресной строке браузера «localhost:8080» будет открыто окно авторизации. Для получения доступа к данным необходимо пройти авторизацию: необходимо ввести логин и пароль, а затем нажать кнопку «Login». На основе прав пользователя будет сформировано меню и доступ пользователя для редактирования таблиц и записей в них. После успешной авторизации пользователь попадает на главную страницу. На главной странице располагается слоган компании.

Общая структура страницы

Шапка

Шапка располагается в верхней части страницы Она содержит надпись с названием предприятия и окно поиска (окно поиска доступно только на страницах с таблицами). При нажатии на надпись «OOO Innovative» пользователь будет перенаправлен на главную страницу сайта. При вводе поискового запроса в строку поиска и нажатии клавиши «Enter» пользователь получает результат поиска по всем полям таблицы, для которой осуществляется поиск.

Меню

В левой части страницы располагается меню. Состав пунктов определяется ролью пользователя в системе. При нажатии на пункт меню пользователь будет перенаправлен на соответствующую страницу сайта. Пункты меню «Сотрудники», «Клиенты», «Команды», «Члены команд» перенаправят пользователя на страницы с соответствующими таблицами. Пункты меню «Тип документа», «Специальность», «Города», «Улицы», «Банки» перенаправят пользователя на страницы с соответствующими таблицами-справочниками.

После выбора пункта меню «Настройки» пользователь будет перенаправлен на экран с настройкой. Чтобы сменить язык в системе, необходимо нажать на кнопку с нужным языком. Доступен выбор русского и английского языков. Чтобы выйти из системы необходимо нажать кнопку «Log out».

После выбора пункта меню «Сменить пароль» пользователь будет перенаправлен на экран со сменой пароля. Для смены пароля необходимо ввести старый пароль, новый пароль и подтвердить новый пароль (ввести новый пароль еще раз). После ввода необходимо нажать на кнопку «Change».

После выбора пункта меню «Документы» пользователь будет перенаправлен на экран с написанием запроса. В поле «Query» нужно написать необходимый SQL-запрос к базе данных. После ввода запроса необходимо нажать клавишу «Enter». Результат запроса будет доступен в таблице. Также можно использовать предзапросы: «Salary» - для получение информации о зарплате сотрудников и «Clients» - для получения данных клиентов предприятия. Чтобы экспортировать результаты запроса в файл необходимо указать назвыание экспортируемого файла в поле «File name» и нажать кнопку «export». Результаты будут сохранены в файле с типом указаном после точке формате.

Таблица

Вверху располагается заголовок текущей таблицы. Далее расположена таблица для просмотра. Чтобы изменить запись в таблице (обновить, удалить) необходимо нажать на нужную запись. Чтобы добавить запись в таблицу необходимо нажать на кнопку «Add record». Обновление и добавление записи доступно, если у пользователя есть права на добавление или обновления соответствующей таблицы.

Чтобы создать новую запись, необходимо заполнить форму с нужными значениями. Выбор некоторых значений доступен из выпадающего списка. Выбор даты доступен при помощи выпадающего календаря. После заполнения формы корректными значениями необходимо нажать кнопку «Update».

Чтобы обновить запись, необходимо заполнить форму нужными значениями и нажать кнопку «Update». Для удаления записи необходимо нажать кнопку «Delete».
